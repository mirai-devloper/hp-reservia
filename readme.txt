== Reservia Reviews for HairsPress ==
Requires as least: 4.7
Tested up to: 5.3
Requires PHP: 5.6.4

== Description ==

Reserviaの予約システムから口コミを取得し表示するプラグインです。

使い方は、シンプルに設定からReservia連携ページへ行き、リザービア番号を入れて保存します。
ショートコードが上部に表示されているので、表示したいページに貼り付けて完了です。
